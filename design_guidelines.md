# Spirit Emeraude Création - Design Guidelines

## Design Philosophy
**Luxury African Artisan Brand** - Premium minimalism that breathes elegance, authenticity, and social impact. Inspired by Mawu Africa's aesthetic: abundant whitespace, clean edges, no heavy shadows, refined premium feel.

## Design System: "Luxe & Terre"

### Typography
- **Headings**: 'Domine' (serif) - Elegant, authoritative
- **Body Text**: 'Montserrat' (sans-serif) - Modern, highly readable
- Use uppercase sparingly for impact (buttons, labels)

### Color Palette
- **Primary**: `#064e3b` (Deep Emerald Green - Luxury)
- **Accent**: `#d4af37` (Muted Gold/Sand - Pagne/Sunlight)
- **Background**: `#fdfbf7` (Eggshell White - Natural paper)
- **Text**: `#292524` (Warm Anthracite Gray)
- **Dark Sections**: `#1f2937` (For footer)

### Spacing & Layout
Use generous whitespace throughout. Employ modern spacing units (4px, 8px, 16px, 24px, 32px, 48px, 64px) for consistent rhythm.

### UI Components
- **Buttons**: Rectangular, uppercase text, thin borders, hover effect with color fill transition
- **Cards**: Ultra-clean, image-focused, minimal decorative elements
- **Images**: High-quality, ratio 3:4 vertical for products, full-bleed where appropriate

## Global Layout

### Header (Sticky)
- Slightly transparent background with glassmorphism effect
- Centered logo "SPIRIT EMERAUDE"
- Desktop: Clean menu items flanking the logo
- Mobile: Burger menu icon
- Subtle shadow on scroll

### Footer
- Dark background (#1f2937)
- Three-column layout: Brand Mission | Quick Links | Contact (Brazzaville)
- Dynamic copyright year
- White/light text for contrast

## Page-Specific Designs

### Home Page

**Hero Section - Split Screen**
- **Left Half**: 
  - Compelling headline: "L'art qui libère..." (large Domine font)
  - Subtitle explaining the mission
  - Prominent CTA button "Découvrir"
- **Right Half**: 
  - Large, high-quality image of a handcrafted pagne bag
  - Full-height, no padding

**Mission/Storytelling Section**
- Clean white background
- Two-column layout: Portrait photo of founder (Emeraude) | Mission text
- Elegant typography hierarchy
- Comfortable reading width for text

**Categories Section**
- Asymmetric Bento Grid layout
- Three featured categories: "Sacs & Pochettes", "Sandales", "Formations"
- Mix of image sizes for visual interest
- Category names overlaid on images with semi-transparent backgrounds

### Shop/Boutique Page

**Header**
- Simple, elegant title: "La Collection"

**Filters**
- Pill-shaped category buttons (Sacs, Sandales, Accessoires)
- Active state with primary color fill

**Product Grid**
- Clean, minimalist cards
- 3:4 vertical image ratio
- Product name below image (subtle, not bold)
- Price in accent color (#d4af37)
- "Commander" button that maintains elegance
- Grid: 3 columns desktop, 2 tablet, 1 mobile

### Formations Page

**Introduction**
- Inspiring text about empowerment and learning
- Centered, comfortable reading width

**Workshop List**
- Vertical list or wide horizontal cards
- Each card displays:
  - Workshop title (Domine font)
  - Duration, Price, Materials provided/required
  - Next session date
  - "S'inscrire" CTA button

### Impact Social Page

**Editorial Style Layout**
- Alternating image-left/text-right sections
- Large, impactful images from workshops
- Descriptive text about humanitarian actions
- Focus on: Orphelinat Sainte Thérèse initiatives
- Respectful photography (no recognizable children's faces)

### Gallery Page

**Masonry Grid** (Pinterest-style)
- Mix of workshop photos, product shots, event images
- Category filters: Atelier, Création, Humanitaire
- Hover effect: subtle overlay with photo title
- Responsive columns: 4 desktop, 3 tablet, 2 mobile

### Contact Page

**Two-Column Layout**

**Left Column**: Company Information
- Phone number
- WhatsApp link
- Email
- Physical location in Brazzaville
- Elegant iconography

**Right Column**: Contact Form
- Fields: Name, Phone, Email, Subject, Message
- Clean, minimal input styling
- Prominent submit button
- Success notification on submission

## Interaction Design

### WhatsApp Integration
- Product "Commander" buttons: Open WhatsApp with pre-filled message: "Bonjour Spirit, je souhaite commander le [Product Name] à [Price] FCFA."
- Formation "S'inscrire" buttons: Pre-filled message: "Bonjour, je suis intéressé par la formation [Formation Name]."

### Responsive Behavior
- Mobile-first approach
- Burger menu for mobile navigation
- Stack columns on smaller screens
- Maintain generous spacing across all breakpoints

## Images

### Required Images
1. **Hero**: High-quality photo of luxury pagne handbag (right side split-screen)
2. **Mission**: Portrait of founder Emeraude
